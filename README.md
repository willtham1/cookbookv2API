# cookbookv2 API
This is the back-end API for the cookbookv2. This is a RESTful API with CRUD operations.

## Tech Stack
- Node.js as our runtime environment
- NestJS our Node.js framework
- Docker for containerizing our database
- PostgreSQL
- Prisma

##
Launch Project
```bash
npm install
npm run dev:start
```
Launch backend
```docker
docker-compose up -d
```
